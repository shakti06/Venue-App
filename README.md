# Venue App

This project is simple venue app based on the venue of the musical artist and the application developed using the react and its dependencies.

## Dependencies

- react js

- react-slick

- react-reveal

## Available Scripts

> npm start 

- to start the project

> npm run

- to run the projects

> npm build

- to build the project
